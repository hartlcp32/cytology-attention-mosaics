Cytology Attention Map Mosaics

Overview

These mosaics visualize GradCAM attention patterns from a ResNet50 model trained on cervical cytology cell classification (SIPaKMeD dataset, 4,049 cells, 5 classes).

Key Finding: The model achieves 96.4% classification accuracy, but directs 31.8% of high-confidence attention to regions OUTSIDE the cell boundary.

Methods

Model: ResNet50 trained on SIPaKMeD cropped cell images (128x128 pixels)

Attention Maps: GradCAM applied to final convolutional layer (layer4)

High Attention Threshold: ≥0.75 normalized GradCAM intensity

Cell Boundaries: Pathologist annotations from SIPaKMeD dataset
